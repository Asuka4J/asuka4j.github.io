``` python 直接选择排序(python)
"""\
    选择排序
    L: 待排序列
"""
def selection(L):
    if not L: return []
    for x in range(0, len(L)-1):
        minIndex = x
        for y in range(x+1, len(L)-1):
            if L[y] < L[minIndex]: minIndex = y
        tmp = L[x]
        L[x] = L[minIndex]
        L[minIndex] = tmp
    return L

```
